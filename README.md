# Servo-Motor-Control-using-Raspberry-Pi
Servo motor control using Raspberry Pi and PWM signals to rotate the motor to different angles. The system uses GPIO to generate precise duty cycles, making it useful for robotics, automation, and embedded control applications.

## Description

This project demonstrates how to control a servo motor using Raspberry Pi. The servo motor is operated using PWM (Pulse Width Modulation) signals generated through GPIO pins, allowing it to rotate to different positions.

## Components Used

* Servo Motor Module
* Jumper Wires
* Universal IoT Trainer Kit with raspberry pi
* USB Cables
* Connecting Wires

## Software Used

* Thonny Python IDE
* Raspbian OS

## Connections

* Servo Signal → GPIO Pin 17 (BCM mode)
* Servo VCC → 5V
* Servo GND → GND

## Circuit Diagram Explanation

The servo motor is controlled using PWM signals from the Raspberry Pi. By changing the duty cycle of the PWM signal, the position of the servo motor shaft is adjusted to different angles.

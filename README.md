# Servo Motor Control using Raspberry Pi

##  Description
This project demonstrates controlling a servo motor using PWM signals generated from Raspberry Pi GPIO pins. The servo rotates to different angles based on the duty cycle.

##  Components Used
- Servo Motor Module
- Jumper Wires
- Universal IoT Trainer Kit with raspberry pi
- USB Cables
- Connecting Wires

##  Connections
- Signal → GPIO 17  
- VCC → 5V  
- GND → GND  

##  Working Principle
The Raspberry Pi generates PWM signals to control the position of the servo motor. By adjusting the duty cycle, the motor shaft rotates to specific angles.
